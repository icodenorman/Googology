Up Arrow

We all know how Multiplication is just repeated Addition, and that Exponation is just repeated Multiplication, but what is repeated Exponation? This is where Knuth’s Up Arrow Notation comes in, Knuth’s Up Arrow Notation allows you to perform “Hyperoperations” one example of this is 22222 this is a big number but there is another way to write it you can write it like this 2^^5. This is known as Tetration, but what about repeated tetration? Pentation of course! In Expanded form this is 2^^2^^2^^2^^2 but using Up Arrow Notation you can write it as 2^^^5 and you can go even further to Hexation where 2^^^2^^^2^^^2^^^2 can be written as 2^^^^2, since we do not have anymore words for bigger numbers we will write them out like this 2^52 this is equal to 2^^^^^2, let’s take this to the extreme. Start with 3^^^^3, already this is a massive number impossible to write out but we’ll just say this answer g0 (3^^^^3 = g0). Now Hyperoperate 3 to g0 (3^g03) or two 3’s with g0 arrows, we can say this answer is g1 (3^g03 = g1). Now do the same thing again (3 Hyperoperated to g1), you can say this answer is g2 keep repeating the process till you reach g64 This number is incomprehensibly big and the main last number in this series so usually people capitalize it to G64. This is not the end though, you can take it farther to numbers like g65 or even gGoogol or, heck even gG64
